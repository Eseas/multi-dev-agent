**지금 즉시 다음 작업을 수행하세요:**

### 1단계: Write/Edit 도구로 파일 작성

모든 필요한 파일을 **실제로 작성**하세요. 설명하지 말고 작성하세요!

**예시 - 새 파일:**
```
Write 도구 사용:
- file_path: "module-admin/src/main/java/com/dailyword/admin/domain/Admin.java"
- content: 실제 Java 코드 전체
```

**예시 - 기존 파일 수정:**
```
Edit 도구 사용:
- file_path: "gateway/src/main/java/config/SecurityConfig.java"
- old_string: 실제 기존 코드
- new_string: 수정된 코드
```

### 2단계: work-done.md 작성

**반드시** `work-done.md` 파일을 작성하세요:

```markdown

## 구현 요약
- 3-5줄로 무엇을 구현했는지

## 생성/수정된 파일
- 파일경로: 설명
- 파일경로: 설명

## 실행 방법
\`\`\`bash
실행 명령어
\`\`\`

## 기술적 결정
- 주요 결정 사항

## 알려진 제한사항
- 미구현 부분이나 개선점
```

### 3단계: Git 커밋

```bash
git add .
git commit -m "구현: [기능명]"
```
